`nuxt-rellax` wraps the battle-tested [Rellax.js](https://dixonandmoe.com/rellax/){target="_blank"} parallax library into a Nuxt module. It provides three ways to add parallax effects ŌĆö a directive, a composable, and a component ŌĆö all SSR-safe and fully typed.


## Features

:::prose-icon-list{variant="success"}
::li
**`v-rellax` directive** ŌĆö drop parallax onto any element declaratively, with optional modifier shortcuts
::
::li
**`useRellax` composable** ŌĆö full programmatic control with `refresh`, `destroy`, and `reinitialize`
::
::li
**`useRellaxGroup` composable** ŌĆö manage a batch of parallax elements together
::
::li
**`<RellaxElement>` component** ŌĆö declarative wrapper with typed props and reactive prop changes
::
::li
**`prefers-reduced-motion` support** ŌĆö respects the user's OS motion preference out of the box
::
::li
**Responsive disable** ŌĆö optionally disable parallax below a configurable viewport width
::
::li
**Intersection Observer mode** ŌĆö lazy-init elements only when they enter the viewport
::
::li
**Module-level defaults** ŌĆö set global Rellax options once in `nuxt.config.ts`
::
:::

## Installation

:::prose-steps
::prose-step

### Install the package

:prose-pm-install{name="nuxt-rellax"}

::
::prose-step

### Register the module

Add `nuxt-rellax` to your `modules` array in `nuxt.config.ts`:

```ts [nuxt.config.ts] noFormat
export default defineNuxtConfig({
  modules: ['nuxt-rellax'],
})
```

::
::prose-step

### Add your first parallax element

Use the `v-rellax` directive on any element. Make sure the element has room to scroll ŌĆö it needs to be inside a scrollable page or container.

```vue [pages/index.vue]
<template>
  <div class="h-[200vh]">
    <div v-rellax="{ speed: -3 }" class="fixed top-1/2 left-1/2">
      <img src="/hero-bg.jpg" alt="Parallax background" />
    </div>
  </div>
</template>
```

::
:::

## Quick Examples

### Directive

The simplest way to add parallax ŌĆö just attach `v-rellax` to any element.

```vue [HeroSection.vue]
<template>
  <!-- Slow drift upwards -->
  <div v-rellax="{ speed: -2 }">Background layer</div>

  <!-- Fast drift with centering -->
  <div v-rellax.center="{ speed: -5 }">Foreground layer</div>

  <!-- Horizontal parallax -->
  <div v-rellax.horizontal="{ speed: 3 }">Side element</div>
</template>
```

### Composable

Use `useRellax` when you need programmatic control or want to target elements by selector.

```vue [ParallaxSection.vue]
<script setup lang="ts">
const elRef = useTemplateRef<HTMLElement>('bg')

const { refresh, reinitialize } = useRellax(elRef, { speed: -3, center: true })
</script>

<template>
  <div ref="bg" class="parallax-bg">
    <slot />
  </div>
</template>
```

### Component

`<RellaxElement>` exposes all Rellax options as typed props, and calls `reinitialize` automatically when any prop changes.

```vue [Hero.vue]
<template>
  <div class="hero">
    <RellaxElement :speed="-2" :center="true" class="hero-bg">
      <img src="/hero.jpg" alt="" />
    </RellaxElement>

    <RellaxElement tag="h1" :speed="1">
      Welcome
    </RellaxElement>
  </div>
</template>
```
